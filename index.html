Monitoramento em Tempo Real

O dashboard exibe os dados coletados pelo ESP32 durante o processo de produção da cerveja, permitindo visualizar em tempo real a estabilidade térmica do sistema.

O monitoramento contínuo garante maior precisão no controle da mosturação e fervura, reduzindo oscilações de temperatura.

Interpretação dos Dados

O gráfico acompanha a evolução térmica do processo cervejeiro, demonstrando como o sistema reage às variações de temperatura.

O controle ON-OFF com histerese permite estabilizar a faixa térmica, evitando acionamentos excessivos do relé e aumentando a eficiência do sistema.

Resumo do Projeto

O Sonic Boom Bar é um sistema automatizado desenvolvido para auxiliar no controle térmico da produção artesanal de cerveja.

O projeto integra eletrônica, programação, IoT e automação, utilizando um ESP32 como núcleo de processamento.

Objetivos do Projeto

  • Automatizar o controle térmico da brassagem
  • Monitorar temperatura em tempo real
  • Garantir estabilidade térmica de ±0,5°C
  • Permitir acompanhamento remoto via Wi-Fi
  • Aplicar conceitos de IoT e sistemas embarcados

Funcionamento do Circuito

O sistema utiliza sensores DS18B20 para medir a temperatura do mosto, enquanto o ESP32 processa os dados e aciona o módulo relé responsável pelo aquecimento.

O controle ocorre através de lógica ON-OFF com histerese, mantendo a temperatura dentro da faixa desejada.

Processo da Cerveja

Ingredientes

  • Malte Pilsen
  • Lúpulo
  • Levedura Lager
  • Água

Processo

  1. Moagem do malte
  2. Mosturação entre 62°C e 68°C
  3. Filtragem do mosto
  4. Fervura a aproximadamente 100°C
  5. Resfriamento
  6. Fermentação entre 8°C e 12°C
  7. Maturação entre 0°C e 4°C
